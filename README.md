# Human-Resource-Management-System
A console-based Human Resource Management System (HRMS) built using Core Java that manages employee records, attendance, leave management, and reporting with authentication and search functionality.



##  Overview
A Java console-based HRMS application for managing employee data efficiently.

---

##  Features
- Add / Update / Delete Employees
- Attendance Tracking
- Leave Management (Approve/Reject)
- Employee Search
- Report Generation
- Login Authentication

---

##  Technologies Used
- Java (Core)
- ArrayList / File Handling
- OOP Concepts

---

##  Run
```bash
javac HRMS.java
java HRMS
